# HRMS Lite

A lightweight Human Resource Management System to manage employees and attendance.

---

## Tech Stack

Frontend:

- React (Vite)
- Axios

Backend:

- FastAPI
- MongoDB
- PyMongo

Deployment:

- Frontend: Vercel
- Backend: Render

---

## Features

- Add employee
- Delete employee
- Mark attendance
- View attendance per employee
- Server-side validation
- Proper error handling
- RESTful API design

## Run Locally

Backend:

cd backend  
pip install -r requirements.txt  
uvicorn app.main:app --reload

Frontend:

cd frontend  
npm install  
npm run dev

---

## Assumptions

- Single admin user
- No authentication required
- Basic HR functionality only (lite system)
